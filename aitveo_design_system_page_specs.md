# AITVEO Website Design.md

## Design Philosophy
This design should reflect a **senior, enterprise-grade UI/UX approach** — minimal, confident, structured, and outcome-driven.

**Style Direction:**
- Apple-level minimalism + Accenture/Palantir enterprise clarity
- Strong typography hierarchy
- Clean grids, lots of whitespace
- Subtle motion (not flashy)
- Dark + light hybrid (primary: light with deep accents)

**Core Feel:**
Premium • Intelligent • Scalable • Trustworthy

---

## Brand Usage
- Use provided AITVEO logo (header + footer)
- Maintain generous padding around logo
- Prefer monochrome or subtle gradient backgrounds

---

## Color System (Suggested)
- Primary: Deep Navy (#0B1F3B)
- Accent: Electric Blue / Teal Gradient
- Background: #FFFFFF / Soft Gray (#F7F9FC)
- Text Primary: #0A0A0A
- Text Secondary: #5A6B7B

---

## Typography
- Headings: Modern sans-serif (Inter / SF Pro / Helvetica Now)
- Body: Clean readable sans-serif
- Large bold hero headings
- Tight, consistent spacing

---

## GRID & SPACING
- 12-column grid
- Max width: 1200–1320px
- Section spacing: 80–120px vertical
- Consistent alignment across all pages
